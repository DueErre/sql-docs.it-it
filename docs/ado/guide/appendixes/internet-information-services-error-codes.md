|------------------------------------------|------------|-----------------|

|**IDS_IIS_ObjectNotFound**|8209-2146820079 0x800A2011|Errore del Server Internet: Modulo o l'oggetto non trovato.|
|**IDS_IIS_RequestForbidden**|8210 materiale-2146820078 0x800A2012|Errore del Server Internet: Richiesta non è consentito.|
|**IDS_IIS_UnexpectedError**|8447-2146819841 0x800A20FF|Errore del Server Internet.|
