# <a name="position-object-ado-md"></a>Oggetto Position (ADO MD)
Rappresenta un set di uno o più membri di dimensioni diverse che definisce un punto lungo un asse.  
  
## <a name="remarks"></a>Osservazioni  
 Con le proprietà e raccolte di un **posizione** dell'oggetto è possibile eseguire le operazioni seguenti:  
  
-   Utilizzare il **ordinale** proprietà per restituire la posizione ordinale del **posizione** lungo il [asse](../../../ado/reference/ado-md-api/axis-object-ado-md.md).  
  
-   Utilizzare il [membri](../../../ado/reference/ado-md-api/members-collection-ado-md.md) insieme per restituire i membri che costituiscono la posizione lungo la **asse**.
