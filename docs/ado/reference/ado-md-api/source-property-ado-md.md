# <a name="source-property-ado-md"></a>Proprietà Source (ADO MD)
Indica l'origine per i dati di [set di celle](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Restituisce o imposta un **Variant**, e lettura/scrittura per chiuso [set di celle](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) oggetti e di sola lettura per aprire **set di celle** oggetti. Il **Variant** deve contenere un valore valido **stringa**, ad esempio, una query MDX.
