# <a name="type-property-column-adox"></a>Proprietà Type (colonna) (ADOX)
Indica il tipo di dati di una colonna.  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Restituisce o imposta un **lungo** valore che può essere uno del [DataTypeEnum](../../../ado/reference/ado-api/datatypeenum.md) costanti. Il valore predefinito è **adVarWChar**.  
  
## <a name="remarks"></a>Osservazioni  
 Questa proprietà è di lettura/scrittura finché il [colonna](../../../ado/reference/adox-api/column-object-adox.md) oggetto viene aggiunto a un insieme o a un altro oggetto.
