# <a name="how-to-create-a-resource-pool-for-python"></a>Come creare un Pool di risorse per Python

In questa sezione è in fase di sviluppo.

> [!NOTE]
> Supporto Python è una nuova funzionalità di SQL Server 2017 ed è in versione provvisoria. Per ulteriori informazioni appena, vedere.

In generale, è possibile monitorare qualsiasi script esterni, inclusi uno che esegue Python, usando lo stesso framework fornito per la governance delle risorse di script R in SQL Server 2016. Per ulteriori informazioni, vedere [governance delle risorse per R](../r/resource-governance-for-r-services.md).
