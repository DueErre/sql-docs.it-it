# <a name="delete-a-power-pivot-data-feed-library"></a>Eliminare una libreria di feed di dati PowerPivot
  Per eliminare una libreria di feed di dati è necessario essere il proprietario del sito. Se nella libreria sono contenuti elementi, saranno eliminati insieme alla libreria.  
  
 È necessario conoscere il nome della libreria di feed di dati che si desidera eliminare.  
  
### <a name="delete-a-data-feed-library"></a>Eliminare una libreria di feed di dati  
  
1.  Fare clic su **Azioni sito**.  
  
2.  Fare clic su **Impostazioni sito**.  
  
3.  In Amministrazione sito fare clic su **Raccolte ed elenchi del sito**.  
  
4.  Fare clic su **Personalizza "\<nome della libreria >"**  
  
5.  In Autorizzazioni e gestione fare clic su **Elimina la raccolta**.
