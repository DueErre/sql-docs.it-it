# <a name="properties-ssas-tabular"></a>Proprietà (SSAS tabulare)
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]Progetti di modello tabulare in [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] dispongono di varie proprietà che definiscono i comportamenti per il progetto, modello, creazione di report e la distribuzione. Le impostazioni delle proprietà sono archiviate in formato XML nel file Model.bim, tuttavia, tutte le proprietà descritte in questa sezione possono essere configurate nelle finestre **Proprietà** in [!INCLUDE[ssBIDevStudio](../../includes/ssbidevstudio-md.md)].
