# <a name="editionid-element"></a>Elemento EditionId
  Contiene il codice identificatore numerico edizione dell'elemento padre [elemento Server (ASSL)](../../../analysis-services/scripting/objects/server-element-assl.md) elemento.  
  
## <a name="syntax"></a>Sintassi  
  
```xml  
  
<Server>  
   ...  
   <EditionId>...</EditionId>  
   ...  
</Server>  
```  
  
## <a name="element-characteristics"></a>Caratteristiche elemento  
  
|Caratteristica|Descrizione|  
|--------------------|-----------------|  
|Tipo di dati e lunghezza|Long|  
|Valore predefinito|Nessuno|  
|Cardinalità|1-1: elemento obbligatorio visualizzato una sola volta.|  
  
## <a name="element-relationships"></a>Relazioni elemento  
  
|Relazione|Elemento|  
|------------------|-------------|  
|Elementi padre|[Elemento Server (ASSL)](../../../analysis-services/scripting/objects/server-element-assl.md)|  
|Elementi figlio|Nessuno|
