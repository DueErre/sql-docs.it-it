# <a name="reserved-database-names"></a>Nomi di Database riservato
Un nome di database riservato è un nome non può essere assegnato a un database utente in SQL Server PDW.  
  
Elenco di nomi di database riservato:  
  
-   master  
  
-   model  
  
-   msdb  
  
-   tempdb  
  
-   pdwtempdb  
  
-   pdwtempdb1  
  
-   pdwtempdb2  
  
-   DWConfiguration  
  
-   DWDiagnostics  
  
-   DWQueue
