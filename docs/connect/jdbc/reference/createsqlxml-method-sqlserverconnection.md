# <a name="createsqlxml-method-sqlserverconnection"></a>Metodo createSQLXML (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un **SQLXML** oggetto senza dati.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Un oggetto SQLXML.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo createSQLXML viene specificato tramite il metodo createSQLXML nell'interfaccia Java.SQL. Connection.
