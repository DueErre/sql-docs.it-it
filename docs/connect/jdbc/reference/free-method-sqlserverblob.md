# <a name="free-method-sqlserverblob"></a>Metodo free (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Questo metodo libera l'oggetto BLOB e rilascia le risorse bloccate.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public void free()  
```  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo libero viene specificato dal metodo nell'interfaccia Java.SQL. BLOB disponibile.
