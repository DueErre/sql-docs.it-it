# <a name="getasciistream-method-sqlservercallablestatement"></a>Metodo getAsciiStream (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come flusso di **ASCII** caratteri.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[int getAsciiStream &#40; &#41;](../../../connect/jdbc/reference/getasciistream-int.md)|Recupera il valore del parametro designato come flusso di **ASCII** caratteri in base all'indice del parametro.|  
|[getAsciiStream &#40;java.lang.String &#41;](../../../connect/jdbc/reference/getasciistream-java-lang-string.md)|Recupera il valore del parametro designato come flusso di **ASCII** caratteri in base al nome del parametro.|
