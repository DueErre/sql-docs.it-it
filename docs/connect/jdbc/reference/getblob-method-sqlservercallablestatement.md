# <a name="getblob-method-sqlservercallablestatement"></a>Metodo getBlob (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro BLOB JDBC designato come oggetto Blob nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int.md)|Recupera il valore del parametro BLOB JDBC designato come oggetto Blob nel linguaggio di programmazione Java specificato all'indice del parametro.|  
|[getBlob (lang)](../../../connect/jdbc/reference/getblob-method-java-lang-string.md)|Recupera il valore del parametro BLOB JDBC designato come oggetto Blob nel linguaggio di programmazione Java specificato il nome del parametro.|
