# <a name="getcharacterstream-method-sqlserverclob"></a>Metodo getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il **Clob** valore designato dall'oggetto java.SQL. Clob come oggetto java.IO. Reader.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[Metodo getCharacterStream &#40; &#41;](../../../connect/jdbc/reference/getcharacterstream-method.md)|Restituisce i dati Clob come oggetto java.io.Reader o come flusso di caratteri.|  
|[getCharacterStream metodo &#40; long, long &#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Restituisce i dati Clob come oggetto java.IO. Reader o come un flusso di caratteri con la posizione specificata e la lunghezza.|
