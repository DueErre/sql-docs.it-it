# <a name="getint-method-sqlservercallablestatement"></a>Metodo getInt (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **int** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getInt (int)](../../../connect/jdbc/reference/getint-method-int.md)|Recupera il valore del parametro designato come un **int** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getInt (lang)](../../../connect/jdbc/reference/getint-method-java-lang-string.md)|Recupera il valore del parametro designato come un **int** nel linguaggio di programmazione Java specificato il nome del parametro.|
