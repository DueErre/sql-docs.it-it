# <a name="getjdbcmajorversion-method-sqlserverdatabasemetadata"></a>Metodo getJDBCMajorVersion (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il numero della versione principale di JDBC per questo driver.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public int getJDBCMajorVersion()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Un **int** che indica la versione principale di JDBC.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo getJDBCMajorVersion viene specificato dal metodo getJDBCMajorVersion nell'interfaccia DatabaseMetaData.
