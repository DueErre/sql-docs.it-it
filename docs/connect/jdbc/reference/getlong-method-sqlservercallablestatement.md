# <a name="getlong-method-sqlservercallablestatement"></a>Metodo getLong (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **lungo** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getLong (int)](../../../connect/jdbc/reference/getlong-method-int.md)|Recupera il valore del parametro designato come un **lungo** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getLong (lang)](../../../connect/jdbc/reference/getlong-method-java-lang-string.md)|Recupera il valore del parametro designato come un **lungo** nel linguaggio di programmazione Java specificato il nome del parametro.|
