# <a name="getref-method-sqlserverresultset"></a>getRef (metodo) (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore della colonna designata nella riga corrente di questo [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) oggetto come oggetto di riferimento nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getRef (int)](../../../connect/jdbc/reference/getref-method-int-sqlserverresultset.md)|Recupera il valore di indice della colonna designata nella riga corrente di questo [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) oggetto come oggetto di riferimento nel linguaggio di programmazione Java.|  
|[getRef (lang)](../../../connect/jdbc/reference/getref-method-java-lang-string-sqlserverresultset.md)|Recupera il valore del nome della colonna designata nella riga corrente di questo [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) oggetto come oggetto di riferimento nel linguaggio di programmazione Java.|
