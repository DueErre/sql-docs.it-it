# <a name="getstring-method-sqlservercallablestatement"></a>Metodo getString (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **stringa** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getString (int)](../../../connect/jdbc/reference/getstring-method-int.md)|Recupera il valore del parametro designato come un **stringa** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getString (lang)](../../../connect/jdbc/reference/getstring-method-java-lang-string.md)|Recupera il valore del parametro designato come un **stringa** nel linguaggio di programmazione Java specificato il nome del parametro.|
