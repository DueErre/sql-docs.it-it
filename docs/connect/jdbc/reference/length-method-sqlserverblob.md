# <a name="length-method-sqlserverblob"></a>Metodo length (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il numero di byte nell'oggetto BLOB.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Oggetto **lungo** valore che specifica il numero di byte.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo di lunghezza viene specificato dal metodo di lunghezza nell'interfaccia Java.SQL. BLOB.
