# <a name="setdatabasename-method-sqlserverdatasource"></a>Metodo setDatabaseName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il nome del database a cui connettersi.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public void setDatabaseName(java.lang.String databaseName)  
```  
  
#### <a name="parameters"></a>Parametri  
 *databaseName*  
  
 Oggetto **stringa** che contiene il nome del database.
