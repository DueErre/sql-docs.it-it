# <a name="setint-method-sqlserverpreparedstatement"></a>Metodo (SQLServerPreparedStatement) setInt
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il parametro designato il dato **int** valore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public final void setInt(int n,  
                         int value)  
```  
  
#### <a name="parameters"></a>Parametri  
 *n*  
  
 Un **int** che indica il numero di parametro.  
  
 *Valore*  
  
 Un **int** valore.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo setInt viene specificato dal metodo setInt nell'interfaccia Java.SQL. PreparedStatement.
