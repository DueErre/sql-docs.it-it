# <a name="updateobject-method-sqlserverresultset"></a>Metodo updateObject (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Aggiorna la colonna designata con un **oggetto** valore.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[updateObject (int, lang)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object.md)|Aggiorna la colonna designata con un **oggetto** valore in base all'indice di colonna.|  
|[updateObject (int, lang, int)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object-int.md)|Aggiorna la colonna designata con un **oggetto** valore base all'indice di colonna e una scala.|  
|[updateObject (lang. String, lang)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object.md)|Aggiorna la colonna designata con un **oggetto** valore in base al nome della colonna.|  
|[updateObject (lang. String, lang, int)](../../../connect/jdbc/reference/updateobject-method-java-lang-string-java-lang-object-int.md)|Aggiorna la colonna designata con un **oggetto** valore specificato il nome di colonna e alla scala.|
