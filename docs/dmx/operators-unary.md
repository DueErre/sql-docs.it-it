# <a name="operators---unary"></a>Operatori - unario
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Gli operatori unari eseguono un'operazione su un singolo operando, ad esempio la restituzione del valore positivo o negativo di un'espressione numerica.  
  
 Gli operatori unari supportati da DMX sono indicati nella tabella seguente.  
  
|Operatore|Description|  
|--------------|-----------------|  
|[-&#40; Negativo &#41; &#40; MDX &#41;](../mdx/negative-mdx.md)|Restituisce l'opposto del valore di un'espressione numerica.|  
|[+ &#40; Positivo &#41; &#40; MDX &#41;](../mdx/positive-mdx.md)|Restituisce il valore positivo di un'espressione numerica.|
