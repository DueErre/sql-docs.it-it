# <a name="convert-function-limitations"></a>CONVERTIRE limitazioni (funzione)
Tipo di risultato di errori di conversione nella colonna interessata viene impostata su NULL.  
  
 Tipo di dati data né TIMESTAMP può essere convertito in un altro dati tipo (o stesso) per la funzione CONVERT.
