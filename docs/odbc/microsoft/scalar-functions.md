# <a name="scalar-functions"></a>Funzioni scalari
I driver Microsoft ODBC Desktop Database supportano le funzioni scalari seguenti:  
  
||||  
|-|-|-|  
|CONCATENA|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|Utente|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 Per informazioni sugli argomenti e valori restituiti delle funzioni scalari, vedere l'appendice F del *riferimento per programmatori ODBC.*
