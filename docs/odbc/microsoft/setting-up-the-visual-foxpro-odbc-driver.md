# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Configurare il Driver ODBC di Visual FoxPro
Per informazioni sul download e installazione del driver ODBC Visual FoxPro, vedere [Driver ODBC Visual FoxPro](http://go.microsoft.com/fwlink/?LinkId=121318) nel download su MSDN.  
  
 Utilizzare il programma di installazione di Microsoft Visual FoxPro ODBC Driver per eseguire le operazioni seguenti:  
  
-   Aggiungere nuovi componenti.  
  
-   Rimuovere i componenti installati.  
  
-   L'installazione per ripristinare file e le impostazioni mancanti.  
  
-   Rimuovere tutti i componenti installati in precedenza.  
  
 Dopo aver installato il driver nel sistema, il programma di installazione riconosce i componenti installati i driver e vengono visualizzate le finestre di dialogo aggiuntive che consentono di modificare la configurazione del driver.
