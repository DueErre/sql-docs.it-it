# <a name="supported-scalar-functions-visual-foxpro-odbc-driver"></a>Funzioni scalari supportate (Visual FoxPro ODBC Driver)
Il Driver ODBC di Visual FoxPro supporta tre tipi di funzioni scalari, come definito nella grammatica SQL ODBC: stringa, numerici e funzioni di data e ora.  
  
 In questa sezione vengono trattati gli argomenti seguenti.  
  
-   [Funzioni stringa (Visual FoxPro ODBC Driver)](../../odbc/microsoft/string-functions-visual-foxpro-odbc-driver.md)  
  
-   [Funzioni numeriche, Driver ODBC di Visual FoxPro,](../../odbc/microsoft/numeric-functions-visual-foxpro-odbc-driver.md)  
  
-   [Funzioni di ora e data (Driver ODBC di Visual FoxPro)](../../odbc/microsoft/time-and-date-functions-visual-foxpro-odbc-driver.md)
