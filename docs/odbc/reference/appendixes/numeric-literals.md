# <a name="numeric-literals"></a>Valori letterali numerici
Quando i valori di dati numerici vengono archiviati nelle stringhe di caratteri, vengono utilizzati valori letterali numerici. Per la conversione di dati numerici di SQL su una stringa SQL_C_CHAR o conversione di dati numerici di C in una stringa SQL_CHAR o SQL_VARCHAR, sintassi del valore letterale numerica viene utilizzata per specificare il contenuto archiviato nel database di destinazione. Per la conversione di numerico archiviato come stringa SQL_C_CHAR a dati numerici di SQL o numerico archiviato sotto forma di stringa a dati C numerici SQL_CHAR, questa sintassi Ã¨ utilizzata per convalidare il contenuto archiviato nell'origine.
