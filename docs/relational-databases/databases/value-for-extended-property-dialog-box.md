# Finestra di dialogo Valore per propriet&#224; estesa
  Usare la finestra di dialogo **Valore per** *\<property name>* per immettere o visualizzare un valore. È possibile accedere a questa finestra di dialogo da diverse posizioni.  
  
## Elenco degli elementi di interfaccia  
 **Nome proprietà estesa**  
 Nome della proprietà estesa visualizzata o impostata.  
  
 **Valore proprietà estesa**  
 Immettere o modificare il valore della proprietà estesa
