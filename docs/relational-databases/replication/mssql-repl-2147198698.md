# MSSQL_REPL-2147198698
    
## Dettagli messaggio  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|-2147198698|  
|Origine evento|MSSQLServer|  
|Nome simbolico||  
|Testo del messaggio|Lo snapshot per questa pubblicazione è diventato obsoleto. Per sincronizzare la sottoscrizione, eseguire di nuovo l'agente snapshot.|  
  
## Spiegazione  
 Lo snapshot per questa pubblicazione è diventato obsoleto.  
  
## Azione dell'utente  
 L'agente snapshot deve essere eseguito di nuovo prima di poter sincronizzare la sottoscrizione.
