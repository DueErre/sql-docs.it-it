# <a name="mssqlrepl-2147200950"></a>MSSQL_REPL-2147200950

|-|-|

|Origine evento|MSSQLServer|  
|Nome simbolico||  
|Testo del messaggio|La funzione di checksum utilizzata dal processo di merge per la convalida dei dati per l'articolo '% 1' ha restituito un valore di checksum non valido. Per risolvere il problema, utilizzare SQL Profiler o riavviare l'agente con un valore di -HistoryVerboseLevel pi√π elevato, quindi individuare eventuali errori nel file del log di output. ## <a name="explanation"></a>Spiegazione  
 Una stored procedure ha restituito un valore NULL o 0 per il checksum. La causa potrebbe essere un errore del server.  
  
## <a name="user-action"></a>Azione dell'utente  
 Cercare altri errori generati dal server. Correggere tutte le condizioni del [!INCLUDE[ssDE](../../includes/ssde-md.md)] che potrebbero aver impedito il completamento dell'operazione di checksum.
