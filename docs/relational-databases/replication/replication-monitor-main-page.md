# Monitoraggio replica, pagina principale
  Monitoraggio replica consente all'utente di tenere traccia dello stato e delle prestazioni di pubblicazioni e sottoscrizioni in una topologia di replica. Per ulteriori informazioni, vedere gli argomenti indicati di seguito.  
  
-   Per una panoramica di monitoraggio replica, vedere [Monitoraggio replica](../../relational-databases/replication/monitor/monitoring-replication-overview.md).  
  
-   Le informazioni contenute nel riquadro a sinistra di Monitoraggio replica si riferiscono ai server di pubblicazione e ai gruppi di server di pubblicazione. Aggiungere uno o più server di pubblicazione a Monitoraggio replica per visualizzare informazioni di pubblicazione e sottoscrizione. Per ulteriori informazioni, vedere [aggiungere e rimuovere i server di pubblicazione da Monitoraggio replica](../../relational-databases/replication/monitor/add-and-remove-publishers-from-replication-monitor.md).
