# Convalida sottoscrizioni
  Utilizzare il **Convalida sottoscrizioni** la finestra di dialogo per specificare che le sottoscrizioni di una pubblicazione transazionale devono essere convalidate alla successiva esecuzione l'agente di distribuzione per ogni sottoscrizione. I risultati della convalida vengono visualizzati in Monitoraggio replica. Per altre informazioni, vedere [Validate Data at the Subscriber](../../relational-databases/replication/validate-data-at-the-subscriber.md).  
  
## Opzioni  
 **Convalida tutte le sottoscrizioni SQL Server**  
 Selezionare questa opzione per convalidare i dati di tutte le sottoscrizioni SQL Server della pubblicazione specificata.  
  
 **Convalida le sottoscrizioni seguenti**  
 Selezionare questa opzione se non si desidera convalidare tutte le sottoscrizioni nell'apposito elenco.  
  
 **Opzioni di convalida**  
 Fare clic su accesso il **le opzioni di convalida sottoscrizione** nella finestra di dialogo che consente di specificare se utilizzare la convalida mediante conteggio delle righe o convalida mediante checksum binario.
