# <a name="servernetworkprotocol-class"></a>Classe ServerNetworkProtocol
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Il [classe ServerNetworkProtocol](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md) installati in un'istanza di protocolli di rete rappresenta [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] che abilita la comunicazione con i client in rete.  
  
 Con il [classe ServerNetworkProtocol](../../../relational-databases/wmi-provider-configuration-classes/servernetworkprotocol-class/servernetworkprotocol-class.md), è possibile:  
  
-   Controllare nome, versione, file con estensione dll, numero di proprietà e nome visualizzato del protocollo di rete del server a cui si fa riferimento.  
  
-   Fare riferimento a proprietà e indirizzi IP del protocollo di rete.  
  
-   Abilitare e disabilitare il protocollo di rete del server.
