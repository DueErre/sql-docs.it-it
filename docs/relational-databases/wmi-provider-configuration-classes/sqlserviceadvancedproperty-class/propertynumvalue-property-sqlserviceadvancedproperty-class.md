# <a name="propertynumvalue-property-sqlserviceadvancedproperty-class"></a>Proprietà PropertyNumValue (classe SqlServiceAdvancedProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Ottiene il valore numerico della proprietà avanzata.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object.PropertyNumValue [= value]  
```  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore uint32 che specifica il valore numerico della proprietà avanzata.
