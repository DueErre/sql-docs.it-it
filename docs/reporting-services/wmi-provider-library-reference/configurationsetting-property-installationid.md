# <a name="configurationsetting-property---installationid"></a>Proprietà ConfigurationSetting - InstallationID
  Questa proprietà restituisce un identificatore univoco per un'istanza del server di report specifica.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
