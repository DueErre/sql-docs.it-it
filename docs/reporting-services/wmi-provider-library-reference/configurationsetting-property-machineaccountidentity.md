# <a name="configurationsetting-property---machineaccountidentity"></a>Proprietà ConfigurationSetting - MachineAccountIdentity
  Ottiene l'identità dell'account del computer in cui è installato il server di report.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Valore **String** che contiene l'identità dell'account del computer in cui è installato il server di report.
