# <a name="configurationsetting-property---pathname"></a>Proprietà ConfigurationSetting - percorso
  Specifica il percorso di installazione di un'istanza del server di report. Di sola lettura.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Oggetto **String** che rappresenta il percorso di installazione.
