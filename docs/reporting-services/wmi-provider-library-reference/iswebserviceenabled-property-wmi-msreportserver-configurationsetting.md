# Propriet&#224; IsWebServiceEnabled (MSReportServer_ConfigurationSetting WMI)
  Indica se il servizio Web ReportServer è abilitato. Di sola lettura.  
  
## Sintassi  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## Valori della proprietà  
 Valore **booleano** di sola lettura. Un valore **true** indica che il servizio Web ReportServer è abilitato.
