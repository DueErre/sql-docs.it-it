# Propriet&#224; IsWindowsServiceEnabled (MSReportServer_ConfigurationSetting WMI)
  Indica se il servizio Windows ReportServer è abilitato. Di sola lettura.  
  
## Sintassi  
  
```vb  
Public Dim IsWindowsServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWindowsServiceEnabled;  
```  
  
## Valori della proprietà  
 Valore **booleano** di sola lettura. Il valore **true** indica che il servizio Windows ReportServer è abilitato.
