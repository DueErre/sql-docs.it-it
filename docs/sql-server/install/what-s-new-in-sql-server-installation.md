# Novit&#224; relative all&#39;installazione di SQL Server

 L'installazione è supportata solo su processori x64. Per altre informazioni, vedere [Requisiti hardware e software per l'installazione di SQL Server 2016](../../sql-server/install/hardware-and-software-requirements-for-installing-sql-server-2016.md).
  
 L'installazione di [!INCLUDE[ssExpCurrent](../../includes/ssexpcurrent-md.md)] richiede di specificare la directory in cui salvare il pacchetto estratto. Se non viene specificato alcun percorso, verrà usato il valore predefinito del server, che corrisponde all'unità di sistema del computer. I file estratti vengono conservati dopo il completamento dell'installazione di [!INCLUDE[ssExpCurrent](../../includes/ssexpcurrent-md.md)].  
  
 SysPrep è supportato per tutte le installazioni di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. SysPrep ora supporta le installazioni del cluster di failover. Per altre informazioni, vedere [Considerazioni sull'installazione di SQL Server tramite SysPrep](../../database-engine/install-windows/considerations-for-installing-sql-server-using-sysprep.md) e [Installazione di SQL Server tramite SysPrep](../../database-engine/install-windows/install-sql-server-2016-using-sysprep.md).  
  
 L'aggiornamento da [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] è supportato, ma non è supportata l'installazione side\-by\-side. Per altre informazioni sul supporto di [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)] in [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)], vedere [Aggiornamenti di versione ed edizione supportati](../../database-engine/install-windows/supported-version-and-edition-upgrades.md).
