# <a name="proxy-editor---add-principal"></a>Editor proxy - Aggiungi entità
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Usare questa pagina per concedere alle entità di sicurezza del server l'accesso a un account proxy di [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] Agent.  
  
## <a name="options"></a>Opzioni  
**Tipo di entità**  
Tipo di entità da visualizzare.  
  
**Entità disponibili**  
Elenca le entità del tipo selezionato.  
  
**Nome**  
Selezionare il nome dell'entità per concedere l'accesso a tale entità oppure selezionare **Nome** per concedere l'accesso a tutte le entità incluse nell'elenco.
