# <a name="save-changes"></a>Salva modifiche
Avvisa che le modifiche apportate agli elementi elencati non verranno salvate a meno che non si faccia clic su **Sì**.  
  
Questa finestra di dialogo può venire visualizzata alla chiusura di soluzioni, piani di manutenzione o altri file aperti.  
  
### <a name="to-display-the-save-changes-dialog-box"></a>Per visualizzare la finestra di dialogo Salva modifiche  
  
-   Con un nuovo file aperto, scegliere **Chiudi** dal menu **File**.  
  
Le modifiche apportate a elementi salvati in precedenza verranno salvate automaticamente nel relativo percorso di salvataggio all'interno dell'archivio. Se un elemento non è stato ancora salvato, viene visualizzata la finestra di dialogo **Salva file con nome** per richiedere all'utente il nuovo percorso di salvataggio.  
  
## <a name="options"></a>Opzioni  
**Salvare le modifiche apportate ai seguenti elementi?**  
Gli elementi selezionati verranno salvati. È possibile selezionare e deselezionare gli elementi.  
  
> [!NOTE]  
> Durante il salvataggio, specificare solo il nome dell'oggetto, non quello dello schema e dell'oggetto.
