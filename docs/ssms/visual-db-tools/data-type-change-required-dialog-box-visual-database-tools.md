# <a name="data-type-change-required-dialog-box-visual-database-tools"></a>Finestra di dialogo Necessaria modifica al tipo di dati (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Viene visualizzata quando si modifica il tipo di dati, la lunghezza, le proporzioni, la precisione o le regole di confronto di una colonna che fa parte di una relazione.  
  
## <a name="options"></a>Opzioni  
**Sì**  
Modifica la proprietà del tipo di dati delle colonne correlate mostrate nell'elenco in modo che le relazioni esistenti vengano mantenute.  
  
**No**  
Annulla la modifica al tipo di dati e ripristina il tipo di dati precedente nella colonna appena modificata.
