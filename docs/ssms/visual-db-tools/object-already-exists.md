# <a name="object-already-exists"></a>Oggetto già esistente
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] L'oggetto che si cerca di salvare ha lo stesso nome di un oggetto già presente nel database. Scegliere un altro nome e continuare.  
  
Può indicare anche che il nome di colonna esiste già.
