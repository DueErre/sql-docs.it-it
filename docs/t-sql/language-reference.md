# <a name="transact-sql-reference-database-engine"></a>Guida di riferimento a Transact-SQL (Motore di database)
[!INCLUDE[tsql-appliesto-ss2008-all-md](../includes/tsql-appliesto-ss2008-all-md.md)]

In questo argomento fornisce le nozioni di base su come trovare e utilizzare gli argomenti di riferimento di Transact-SQL (T-SQL) di Microsoft. T-SQL è fondamentale per l'utilizzo di servizi e prodotti di Microsoft SQL. Tutti gli strumenti e applicazioni che comunicano con un database SQL è possibile farlo mediante l'invio di comandi T-SQL.  

## <a name="tools-that-use-t-sql"></a>Strumenti che utilizzano T-SQL

Alcuni degli strumenti di Microsoft che rilasciano i comandi T-SQL sono:

- [SQL Server Management Studio](../ssms/download-sql-server-management-studio-ssms.md)
- [SQL Server Data Tools](../ssdt/download-sql-server-data-tools-ssdt.md).  
- [SQLCMD](../tools/sqlcmd-utility.md).  
  
 
## <a name="locate-the-transact-sql-reference-topics"></a>Individuare gli argomenti di riferimento di Transact-SQL  
  
Per trovare gli argomenti di T-SQL, utilizzare la ricerca nella parte superiore di questa pagina o usare il sommario sul lato sinistro della pagina. È anche possibile digitare una parola chiave T-SQL nella finestra dell'Editor di Query di Management Studio e premere F1.

## <a name="applies-to-references"></a>"Si applica a" i riferimenti  
 Gli argomenti di riferimento di T-SQL che includono più versioni di SQL Server, a partire da 2008, nonché gli altri servizi di SQL Azure. Nella parte superiore di ogni argomento è una sezione che indica quali prodotti e servizi supportano l'oggetto dell'argomento. 

Ad esempio, in questo argomento si applica a tutte le versioni e con l'etichetta seguente. 
  
 [!INCLUDE[tsql-appliesto-ss2008-all_md](../includes/tsql-appliesto-ss2008-all-md.md)]   

Un altro esempio, l'etichetta seguente indica un argomento in cui si applica solo a Azure SQL Data Warehouse e Parallel Data Warehouse.

[!INCLUDE[tsql-appliesto-xxxxxx-xxxx-asdw-pdw_md](../includes/tsql-appliesto-xxxxxx-xxxx-asdw-pdw-md.md)]

  
In alcuni casi, l'argomento è utilizzato da un prodotto o servizio, ma tutti gli argomenti non sono supportati. In questo caso alcune sezioni **Si applica a** aggiuntive vengono inserite nelle descrizioni di argomento appropriate nel corpo dell'argomento.
