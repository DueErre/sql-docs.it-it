# NS$&lt;nome servizio&gt; Propriet&#224; (scheda Avanazate)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] viene implementato come servizio denominato in base al formato **NS$***<nome_configurato_da_utente>*. Se sono state definite proprietà personalizzate, verranno visualizzate in questa scheda con i rispettivi valori. È possibile che questa scheda non contenga proprietà.
