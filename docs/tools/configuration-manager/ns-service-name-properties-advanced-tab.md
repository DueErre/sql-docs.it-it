# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;nome servizio&gt; proprietà (scheda Avanzate)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)]viene implementato come un servizio denominato nel formato **NS$***< user_configured name >*. Se sono state definite proprietà personalizzate, verranno visualizzate in questa scheda con i rispettivi valori. È possibile che questa scheda non contenga proprietà.
