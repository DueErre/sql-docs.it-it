# <a name="sql-full-text-filter-daemon-launcher-advanced"></a>Utilità di avvio del daemon filtri full-text di SQL (Avanzate)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]A partire da [!INCLUDE[ssKatmai](../../includes/sskatmai-md.md)], viene utilizzato il servizio di avvio del Daemon di filtri Full-text SQL [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] full-text. Se si utilizza la ricerca full-text, questo servizio deve essere in esecuzione. Per informazioni sui processi host del daemon di filtri, vedere "Architettura della ricerca full-text" nella documentazione online di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
 Per impostazione predefinita, nella scheda **Avanzate** non vengono visualizzate proprietà. Se sono state definite proprietà personalizzate, verranno visualizzate anche in questa scheda con i rispettivi valori.
