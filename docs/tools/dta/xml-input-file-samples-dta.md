# <a name="xml-input-file-samples-dta"></a>Esempi di file di input XML (DTA)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]In questa sezione contiene i file XML di esempio inpui che è possibile utilizzare con il **dta** strumento da riga di comando. Questo strumento è una delle interfacce utente di Ottimizzazione guidata motore di database, un ulteriore strumento che consente di ottimizzare le prestazioni delle query. Ottimizzazione guidata motore di database analizza gli effetti di un carico di lavoro su uno o più database. Un carico di lavoro è un set di istruzioni [!INCLUDE[tsql](../../includes/tsql-md.md)] eseguite sui database che si desidera ottimizzare. Dopo aver analizzato gli effetti del carico di lavoro, Ottimizzazione guidata motore di database genera un'indicazione per l'aggiunta di indici, viste indicizzate o strategie di partizionamento che consentono di migliorare le prestazioni delle query sui database.
