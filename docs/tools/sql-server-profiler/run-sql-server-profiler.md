# Eseguire SQL Server Profiler
  Per eseguire [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)], dal menu **Start** scegliere **Tutti i programmi**, **Microsoft SQL Server 2014**, **Strumenti per le prestazioni**, quindi fare clic su **SQL Server Profiler**.
