# <a name="ordered-and-unordered-expressions-xquery"></a>Espressioni ordinate e non ordinate (XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  Per impostazione predefinita, la modalità di ordinamento per tutte le operazioni [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] è **ordinati**. Di conseguenza, le sequenze di nodi restituiti da espressioni di percorso e le espressioni FLWOR, senza il **Ordina** clausola, sono nell'ordine del documento.  
  
 Altro **ordinati** e **non ordinata** sintassi descritta nella specifica XQuery non è supportata.
